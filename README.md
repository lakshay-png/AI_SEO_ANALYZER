# Rank Pilot — AI SEO Analyzer

An AI-powered SEO analysis platform built using:

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features
- SEO analysis
- Rank tracking
- Authentication
- Dashboard
- History tracking

## Installation

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run server
```

## Environment Variables

Create `.env` inside server:

```env
MONGODB_URI=your_mongodb_url
JWT_SECRET=your_secret
PORT=5000
```
